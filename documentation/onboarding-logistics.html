28. Joining the BloodLife Team

Welcome, new contributor! We are excited to have you. This page is designed to take you from a curious visitor to a productive team member as quickly as possible.

Note: Skill Sharing. We value learning. If you're new to React or TypeScript, don't worry—the project is a great place to learn!

Your First Hour

  1. Explore: Run the app locally (see Setup Guide) and click through the features.
  2. Read: Skim the Architecture Overview to understand how the app is "wired."
  3. Pick a Task: Look for issues labeled "good first issue" on our GitHub repository.

Our Communication Culture

We believe in radical transparency and kindness. Every Pull Request is a conversation, not a judgment. We use GitHub Discussions and Issues to plan our next steps and help each other when we're stuck.

Mentorship

If you're looking for guidance, feel free to @mention the core maintainers in any issue. We are here to help you grow your skills while you help us save lives.
