02. Setting Up BloodLife Locally

Follow these steps to get a local copy of BloodLife India running on your computer. This guide assumes you have Node.js installed.

Quick Tip: We use Vite for lightning-fast development steps. It builds the project in milliseconds!

Step 1: Clone the Repository

Open your terminal and run the following command to download the code:

git clone https://github.com/KunalGhadge/BloodLife-India-MVP---Blood-Bank-Management-System.git

Step 2: Install Dependencies

Enter the project folder and install the necessary libraries. We recommend using the --legacy-peer-deps flag to avoid version conflicts with React 19:

cd BloodLife-India-MVP---Blood-Bank-Management-System
npm install --legacy-peer-deps

Step 3: Run the Development Server

Start the local server. Once it's running, you can view the app at http://localhost:5173.

npm run dev

Common Troubleshooting

  • Node Version: Ensure you are using Node.js v18 or higher.
  • Permission Errors: On Windows, you might need to run your terminal as Administrator.
  • Port Conflict: If 5173 is busy, Vite will automatically pick another port (check the terminal output).
