27. The Building Blocks of BloodLife

We don't reinvent the wheel. BloodLife stands on the shoulders of giants—open-source libraries and tools that help us build a world-class application with speed and stability.

Strategy: Lean and Mean. We only use libraries that add real value to the user experience or developer productivity.

Our Core Tech Stack

  • Vite: The extremely fast build tool that powers our development experience.
  • Lucide React: A beautiful, consistent icon library that helps us stay visually clean.
  • clsx & tailwind-merge: Tiny utilities that help us manage CSS classes efficiently.
  • date-fns: A lightweight library for handling complex date calculations (like unit expirations).

Security and Updates

We use automated tools like Dependabot to watch over our libraries. If a security vulnerability is found in one of our building blocks, we are notified immediately so we can update and protect our users.

Why These Tools?

Every dependency is chosen because it is widely used, well-maintained, and light on performance. This ensures that BloodLife remains a modern app for years to come.
