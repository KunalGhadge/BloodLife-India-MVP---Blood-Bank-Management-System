21. Putting BloodLife Online

We use **Vercel** to host the live version of BloodLife India. Deployment is automated, meaning that every time we update our code on GitHub, the live website updates itself within seconds.

Live Deployment

How Our Pipeline Works

  • Build Phase: Vercel runs npm run build. This converts our TypeScript and React code into highly optimized HTML, CSS, and JavaScript that any browser can understand.
  • Edge Network: Once built, the app is distributed to Vercel's global network. This ensures that a user in India sees the page just as fast as a user in the US.
  • Atomic Deploys: Every update is its own "Deploy." If something goes wrong with a new version, we can roll back to the previous one with just one click.

Setting Up Your Own Deployment

  1. Connect your GitHub account to Vercel.
  2. Import the BloodLife-India-MVP repository.
  3. Vercel will automatically detect that it's a Vite project and set the correct build settings.
  4. Click Deploy and you're live!
